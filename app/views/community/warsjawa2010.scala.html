Introduction

Warsjawa 2010 only called workshops actually happened to be more kind of a conference. I've spoken my favourite subject which was PlayFramework. I was the first to speak and unfortunately a lot to slow to warm up the public in early Saturday morning.

Fortunately the framework was interesting enough to interest a few people though and I was given few questions about it afterwards.

Abstract

PlayFramework is a young web development framework for Java and Scala. It throws away some fundamental Java world solutions (like Servlets) in advance to provide extremely robust development environment. This presentation answers to couple basic question each developer would ask about a new framework. It shows PlayFramework's MVC fundamentals, dynamic reloading and build system.

After my last talk about PF (Warsaw JUG) I was told to show some more meat next time and so I've tried. I hope this time it was quite satisfying.
